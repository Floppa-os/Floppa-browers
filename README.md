# Запуск
Для запуска Floppa браузера на языке Python выполните следующие шаги. Инструкция разделена на подготовку окружения, создание файлов и непосредственный запуск.

### Шаг 1: Установка Python
Если у вас еще не установлен Python, скачайте его с официального сайта python.org. При установке обязательно отметьте галочку **"Add Python to PATH"**.

### Шаг 2: Установка необходимых библиотек
Браузер использует библиотеку PyQt6 для графического интерфейса и движок WebEngine для отображения сайтов. Откройте терминал (командную строку) и введите следующую команду:

```bash
pip install PyQt6 PyQt6-WebEngine
```

### Шаг 3: Подготовка файлов проекта
Вам нужно создать три отдельных текстовых файла в одной папке. Скопируйте предоставленный ранее код в соответствующие файлы:

1.  **browser_window.py** — здесь должен находиться основной класс `MainWindow` и логика работы вкладок. Убедитесь, что в начале файла добавлены все необходимые импорты (PyQt6.QtWidgets, PyQt6.QtWebEngineWidgets и т.д.).
2.  **browser_logic.py** — файл с функцией `setup_browser_profile`.
3.  **main.py** — файл для запуска программы.

### Шаг 4: Проверка структуры
Ваша папка с проектом должна выглядеть так:
*   `main.py`
*   `browser_window.py`
*   `browser_logic.py`

### Шаг 5: Запуск приложения
Запустите браузер через терминал, находясь в папке с проектом, или с помощью среды разработки (VS Code, PyCharm):

**Через командную строку:**
```bash
python main.py
```

### Возможные проблемы и решения:
*   **Ошибка "ModuleNotFoundError":** Убедитесь, что вы успешно выполнили команду `pip install` из Шага 2.
*   **Белый экран вместо сайтов:** Проверьте наличие интернет-соединения. Также некоторые сайты могут блокировать запросы от программных движков, если не настроен корректный User-Agent (в коде `browser_logic.py` он уже прописан).
*   **Ошибка при импорте:** Если вы используете старую версию Python (ниже 3.7), обновите её, так как PyQt6 требует актуальных версий интерпретатора.
